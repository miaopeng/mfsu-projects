# Guanyun Eagles

### Start project

```bash
yarn start
```

### Build project

```bash
yarn build
```

### Start Graphql Mock Server

```bash
yarn mock
```

then start in test env with mocking data

```bash
yarn start:test
```

### Test

Run all test

```bash
yarn test
```

Run single e2e test file

```bash
yarn test login.e2e.js
```

### Type check

```bash
yarn tsc
```
